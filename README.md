# JS Practice

A website I'm building while learning JavaScript. Instead of keeping every mini project in its own separate file, I'm collecting them all here in one place with a nav bar, so I can jump between them easily and keep adding more as I learn new things.

## What's in it so far

- **Clock** — a live digital clock that shows the current time and day
- **Stopwatch** — start, stop, lap, and reset
- **Calculator** — a working calculator with a live expression preview
- **To-Do List** — add, check off, and delete tasks, with a counter
- **Form Validation** — username, email, password (with a strength meter), confirm password, and a role select, all validated as you type

There's also a dark / light mode toggle, and a footer with links to my LinkedIn and email.

## Why

I'm using this as my main practice space for JS. Every time I learn something new or build a small project, it gets added here instead of staying in its own random file.

## Files

- `index.html` — the page structure
- `style.css` — all the styling
- `main.js` — all the functionality

Still a work in progress — I'll keep updating it as I learn more.
